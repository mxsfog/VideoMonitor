# ПЗ 1. Распознавание показаний счётчиков

## Назначение

ПЗ 1 обрабатывает набор фотографий счётчиков и формирует таблицу с
распознанными значениями. Задание вынесено отдельно от видео-pipeline, потому
что работает с изображениями, а не с видеороликами.

## Реализация

Файл:

```text
src/pz1_counters.py
```

Используются:

- OpenCV для подготовки изображения;
- EasyOCR для распознавания текста;
- pandas и openpyxl для итоговой таблицы.

Для каждого изображения строится несколько вариантов обработки: исходное RGB,
CLAHE, бинаризация Otsu, инверсия и дополнительные режимы повышения контраста.
Результаты сохраняются вместе с визуальным коллажем, чтобы было видно, на каком
варианте OCR сработал лучше.

## Вход

```text
data/counters/
```

Ожидается набор изображений счётчиков.

## Выход

```text
output/pz1/
├── recognized.csv
├── recognized.xlsx
├── summary.md
└── visual/<image_id>/collage.jpg
```

`recognized.csv` и `recognized.xlsx` содержат имя файла, распознанный текст,
уверенность OCR и технические признаки обработки. `summary.md` нужен для
быстрого просмотра результата без открытия таблицы.

## Запуск

```bash
python src/pz1_counters.py
```

## Что показывать

1. Код `src/pz1_counters.py`.
2. Таблицу `output/pz1/recognized.xlsx`.
3. Несколько коллажей из `output/pz1/visual/`.
4. Краткий отчёт `output/pz1/summary.md`.

## Ограничения

- результат зависит от качества фотографии и угла съёмки;
- EasyOCR может ошибаться на бликах, засветах и мелком шрифте;
- итоговые значения требуют ручной проверки, если качество кадра низкое.

# ПЗ 2. Нарезка видео на кадры

## Назначение

ПЗ 2 готовит видеоданные для последующих шагов. Скрипт принимает локальный файл
или ссылку, открывает видео и сохраняет кадры с заданной частотой.

## Реализация

Файл:

```text
src/pz2_slicer.py
```

Поддерживаются:

- локальные файлы;
- прямые HTTP/HTTPS-ссылки;
- ссылки, которые может обработать загрузчик видео.

Нарезка выполняется через OpenCV. Частота задаётся параметром `--fps`.

## Вход

```text
data/videos/<video>.mp4
```

или URL.

## Выход

```text
output/pz2/<video>/
├── frame_000000.jpg
├── frame_000001.jpg
└── ...
```

Имя каталога берётся из имени видео. Кадры дальше используются в ПЗ 3, ПЗ 6 и
ПЗ 7.

## Запуск

```bash
python src/pz2_slicer.py data/videos/example.mp4 --fps 1
```

Пример для ссылки:

```bash
python src/pz2_slicer.py "https://example.com/video.mp4" --fps 1
```

## Что показывать

1. Код `src/pz2_slicer.py`.
2. Каталог `output/pz2/<video>/`.
3. Несколько кадров из результата.
4. Лог с исходным fps, количеством кадров и шагом нарезки.

## Ограничения

- высокая частота `--fps` резко увеличивает объём output;
- длинные видео лучше обрабатывать с `--fps 0.5` или `--fps 1`;
- качество следующих ПЗ напрямую зависит от качества сохранённых кадров.

# ПЗ 3. OCR титров

## Назначение

ПЗ 3 извлекает текст, который виден в кадре: субтитры, надписи, титры и
встроенные текстовые вставки. Результат сохраняется в JSON и SRT.

## Реализация

Файл:

```text
src/pz3_subtitles.py
```

Скрипт читает кадры из ПЗ 2, анализирует нижнюю часть кадра и запускает EasyOCR
для русского и английского языков. После распознавания подряд идущие похожие
фразы объединяются через `rapidfuzz`.

## Вход

```text
output/pz2/<video>/frame_*.jpg
```

## Выход

```text
output/pz3/<video>/
├── frames.jsonl
├── subtitles.json
└── subtitles.srt
```

`frames.jsonl` содержит результаты по отдельным кадрам. `subtitles.json` и
`subtitles.srt` содержат уже объединённые интервалы.

## Запуск

```bash
python src/pz3_subtitles.py \
  --frames output/pz2/example \
  --src-fps 1 \
  --langs ru,en
```

## Что показывать

1. Код `src/pz3_subtitles.py`.
2. `output/pz3/<video>/subtitles.srt`.
3. `output/pz3/<video>/subtitles.json`.
4. Пример строки из `frames.jsonl`.

## Ограничения

- OCR плохо работает на мелком, размытом или декоративном шрифте;
- если текст находится не в нижней части кадра, его можно пропустить;
- SRT-тайминги зависят от частоты кадров, выбранной в ПЗ 2.

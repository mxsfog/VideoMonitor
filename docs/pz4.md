# ПЗ 4. Распознавание речи

## Назначение

ПЗ 4 извлекает аудиодорожку из видео и строит транскрипт речи. Результат
используется в курсовой для поиска текстовых признаков и LLM-классификации.

## Реализация

Файл:

```text
src/pz4_whisper.py
```

Используется `faster-whisper`. Перед распознаванием аудио извлекается через
ffmpeg. Для снижения шума включён VAD-фильтр.

## Вход

```text
data/videos/<video>.mp4
```

## Выход

```text
output/pz4/<video>/
├── transcript.json
├── transcript.srt
└── transcript.txt
```

`transcript.json` содержит сегменты с временем начала и конца, текстом и
служебной информацией. `transcript.srt` нужен для просмотра по таймкодам.

## Запуск

```bash
python src/pz4_whisper.py data/videos/example.mp4 --model tiny --lang ru
```

Для более точного, но медленного распознавания можно выбрать `small`, `medium`
или `large-v3`.

## Что показывать

1. Код `src/pz4_whisper.py`.
2. `output/pz4/<video>/transcript.txt`.
3. `output/pz4/<video>/transcript.srt`.
4. Использование транскрипта в итоговом отчёте курсовой.

## Ограничения

- шум, музыка и несколько говорящих ухудшают качество;
- автоматическое определение языка лучше не использовать, если язык известен;
- на CPU длинные видео обрабатываются заметно медленнее.

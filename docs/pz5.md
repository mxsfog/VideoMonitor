# ПЗ 5. Детекция объектов

## Назначение

ПЗ 5 ищет объекты в видео и сохраняет покадровые детекции. Эти результаты
используются как самостоятельный артефакт и как один из сигналов курсового
скоринга.

## Реализация

Файл:

```text
src/pz5_yolo.py
```

Используется YOLOv8n из пакета `ultralytics`. Видео обрабатывается потоково,
без предварительной выгрузки всех кадров в память. Для ускорения используется
параметр `--every-n`: анализируется каждый N-й кадр.

## Вход

```text
data/videos/<video>.mp4
```

## Выход

```text
output/pz5/<video>/
├── detections.jsonl
└── summary.json
```

Если включена отрисовка, дополнительно сохраняется размеченное видео в каталоге
`output/pz5/<video>_pred/`.

## Запуск

```bash
python src/pz5_yolo.py data/videos/example.mp4 --every-n 5
```

## Что показывать

1. Код `src/pz5_yolo.py`.
2. `summary.json` с топом классов.
3. `detections.jsonl` как поток детекций.
4. Размеченное видео, если оно сохранено.

## Ограничения

- YOLOv8n обучена на общих классах COCO и не закрывает специальные классы без
  дообучения;
- мелкие или частично закрытые объекты могут не распознаваться;
- для длинных видео нужно увеличивать `--every-n`, иначе обработка будет
  медленной.

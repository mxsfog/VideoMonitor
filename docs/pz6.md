# ПЗ 6. Классификация кадров

## Назначение

ПЗ 6 классифицирует кадры из видео с помощью ResNet50. Этот шаг даёт общее
описание сцен и помогает проверить, какие визуальные категории встречаются в
ролике.

## Реализация

Файл:

```text
src/pz6_resnet.py
```

Используется `torchvision.models.resnet50` с весами ImageNet. На вход подаются
кадры из ПЗ 2. Для каждого кадра сохраняется top-k предсказаний.

## Вход

```text
output/pz2/<video>/frame_*.jpg
```

## Выход

```text
output/pz6/<video>/
├── predictions.jsonl
└── summary.json
```

`predictions.jsonl` хранит предсказания по кадрам. `summary.json` содержит
сводку по самым частым top-1 классам.

## Запуск

```bash
python src/pz6_resnet.py output/pz2/example
```

## Что показывать

1. Код `src/pz6_resnet.py`.
2. `output/pz6/<video>/summary.json`.
3. Несколько строк из `predictions.jsonl`.
4. Использование результата в общем описании видео.

## Ограничения

- ImageNet-классы не являются специализированными модерационными категориями;
- ResNet описывает кадр в целом и не заменяет детектор объектов;
- для юридически значимых выводов этот сигнал недостаточен сам по себе.
